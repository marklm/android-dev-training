+ Aktivieren von Debugging am Mobile bzw. Entwicklersystem

 - Mobile: Settings / About Phone / 7x klick auf "Buildnummer" ==> aktiviert die Entwickleroptionen am Mobile
   - Mobile muss via USB am Entwicklerrechner angeschlossen sein
   - USB Debugging aktivieren bei Nachfrage, erlauben das der Rechner sich verbinden darf
 
 - Am Entwicklungsrechner muss möglicherweise ein Treiber installiert werden, ist abhängig vom System.
    - Bei Window 10 und Android Studio musste nichts besonderes gemacht werden :-)
